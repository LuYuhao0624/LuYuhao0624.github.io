This project is still in progress.

This project is about adversarial robustness. In short, for a deep classifier (although it applies to most deep neural networks), we can add an imperceptible perturbation to the input so that the classifier predicts it incorrectly. We are exploring adversarial training, a straightforward but effective way of defense, based on Tradeoff-inspired Adversarial Defense via Surrogate-loss Minimization (TRADES) proposed by H Zhang et al. in [1].

We made three major modifications to TRADES and have obtained non-trivial improvement on MNIST. Below is the evaluation. All metrics are evaluated with respect to the test set. The model used for TRADES is the checkpoint released in their GitHub. You can find ours here. Since we are still tuning hyper-parameters on Cifar10, the table below will be under frequent updates, so we do not bother to release our best checkpoints now. If you want to play with it, feel free to contact me. The metrics of TRADES are taken from literature (see above).

Ours MNIST $97.52$ $96.53$ $99.30$

We use the same setting as TRADES (PGD-40 with 0.01 step size and 0.3 perturbation budget, PGD-20 with 0.003 step size and 0.031 perturbation budget) to evaluate. $i$-PGD indicates $i$ random starts.

To load our model, using the following scripts. You can find small_cnn.py, net_mnist.py and resnet.py in the GitHub of TRADES.

# main.py
from models.small_cnn import *  # for MNIST models
from models.net_mnist import *  # for MNIST models
from models.resnet import *     # for Cifar10 models

model = SmallCNN().to(device)
# if using Cifar10, comment the line above,
# otherwise, comment the line below
model = ResNet18().to(device)

model.load_state_dict(torch.load(model_path))
